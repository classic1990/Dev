คำสั่งเริ่ม Backend:

cd backend && npm start

📋 ขั้นตอนเริ่ม Backend:

1. เปิด Terminal หรือ Command Prompt
2. ไปที่โฟลเดอร์ backend:
cd backend
3. ติดตั้ง dependencies (ถ้ายังไม่ได้ทำ):
npm install
4. เริ่ม backend server:
npm start
5. ตรวจสอบว่า server ทำงานที่ port 3000

ตรวจสอบ Status:

curl http://localhost:3000/health
